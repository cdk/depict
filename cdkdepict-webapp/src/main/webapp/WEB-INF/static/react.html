Run a reaction on the molecules provided as SMILES in the left or right box (depending on direction). By default, the transform is applied only once and the reaction is returned and displayed below, the output products/reactants are copied into the other box.
Normalize molecules or reactions provided as SMILES in the left hand box. By default, the transform is applied exclusively to all non-overlapping matches.
Map reactions provided as SMILES in the left box. The transform is run on the reactants and if the resulting product matches the product provided the atoms are mapped.
